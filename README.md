# General Information

This repository contains Homework 3 for ENVS 193DS (Winter 2026). The assignment includes statistical analysis, data visualization, and scientific communication using environmental and personal datasets.

The project analyzes the relationship between soil salinity and California pickleweed (Salicornia virginica) biomass, updates visualizations from a personal data collection project, develops a creative affective visualization, and includes a statistical critique of figures or tables from a scientific paper.

All analyses and figures are completed in R using a Quarto document with annotated code and written interpretations.
